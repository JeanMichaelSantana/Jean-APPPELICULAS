# Jean-APP-PELICULAS
Aplicación web moderna que consume la API de TMDB para mostrar películas populares. Incluye tarjetas responsivas con póster, título, puntuación y sinopsis, caja de búsqueda dinámica en tiempo real y sistema de paginación. 
Diseño oscuro moderno con excelente contraste, sombras y disposición en cuadrícula de 4 columnas. Desarrollada con HTML5, CSS3, Bootstrap 5 y JavaScript puro usando Vite.
